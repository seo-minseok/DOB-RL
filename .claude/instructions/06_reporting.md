# 06 — 보고서 작성 가이드

## CYCLE_REPORT.md 작성 규칙

에이전트가 자동 생성하는 섹션:
- 섹션 1: 변경점 요약 (Base 또는 이전 Cycle 대비 무엇을 변경했는가)
- 섹션 2: 변경 상세 (수정된 파일 목록 및 각 파일의 변경 내용)
- 섹션 3: 하이퍼파라미터 (이 Cycle에서 사용된 주요 값)
- 섹션 4: 학습 결과 (정량 지표 + figure 파일 경로 목록)
- 섹션 6: 메모리 업데이트 제안 (필요 시)

**사람이 직접 작성하는 섹션:**
- 섹션 5: 관찰 및 교훈 — **에이전트가 빈 섹션으로 예약만 함**

```markdown
<!-- 에이전트 자동 생성: 정량 지표, figure 경로 목록 -->

<!-- 에이전트가 필요 시 자동 생성 -->
```

## experiments_log.md 업데이트 규칙

Cycle 완료 후 `/experiments_log.md`에 한 줄 추가:

```markdown
| Cycle_N | YYYY-MM-DD | base/Cycle_M | 핵심 변경점 요약 | 최종 avg reward | 비고 |
```

- 업데이트는 에이전트가 자동 수행.
- 기존 행은 수정하지 않음 — 항상 새 행 추가.
